Pink Diamond jewellery Design

About the Project

  • 2019
  • QUT
  • Designer

The aim of this task was to be able to integrate both digital and analogue forms to create a visual composition. We were given the task of choosing a word, and creating a piece that would represent this. The workd I had chosen was that of Luxury. Thus, I decided to design a jewellery magazine advertisement.
